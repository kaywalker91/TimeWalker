# PRD: TimeWalker - 시간을 걷는 자

## 1. 제품 개요

### 1.1 제품명
**TimeWalker: Echoes of the Past**
- 한국명: 타임워커: 역사의 메아리

### 1.2 비전
전 세계의 역사를 **지도 탐험**과 **시대 여행**을 통해 배우는 인터랙티브 교육 어드벤처 게임. 플레이어는 "시간을 걷는 자(TimeWalker)"가 되어 각 지역의 다양한 시대를 방문하고, 역사적 인물들과 대화하며 살아있는 역사를 경험한다.

### 1.3 핵심 컨셉
```
┌─────────────────────────────────────────────────────────────┐
│                    TimeWalker 핵심 경험                      │
├─────────────────────────────────────────────────────────────┤
│  "공부하는 게임이 아닌, 모험하다 보니 배우는 게임"              │
│                                                             │
│  플레이어 = 시간의 목격자 (Observer) + 참여자 (Participant)   │
│  - 역사를 바꾸지는 않지만                                     │
│  - 역사를 해석하고 이해하는 방식은 달라진다                     │
└─────────────────────────────────────────────────────────────┘
```

### 1.4 차별점
| 기존 역사 교육 앱 | TimeWalker |
|------------------|------------|
| 텍스트 중심 암기 | 대화 기반 스토리텔링 |
| 연대표 나열 | 지도 + 시대 탐험 |
| 일방향 정보 전달 | 선택지 기반 인터랙션 |
| 시험 준비 목적 | 모험/발견의 즐거움 |

### 1.5 목표
**단기 (출시 후 3개월)**
- 앱스토어/플레이스토어 교육 카테고리 출시
- MAU 15,000명 달성
- 스토어 평점 4.5 이상

**장기 (1년)**
- 5개 지역 × 각 3~5개 시대 콘텐츠 완성
- 학교/교육기관 B2B 라이선스 모델 구축
- 글로벌 다국어 지원 (10개 언어)

## 2. 타겟 사용자

### 2.1 주요 세그먼트

| 세그먼트 | 연령 | 특성 | 니즈 |
|---------|------|------|------|
| **1차: 학생** | 10-18세 | 역사 과목 학습, 호기심 많음 | 재미있게 역사 공부, 시험 대비 |
| **2차: 성인 학습자** | 19-35세 | 교양 목적, 자기계발 | 가볍게 역사 상식 습득 |
| **3차: 학부모/교사** | 30-50세 | 자녀/학생 교육 | 신뢰할 수 있는 교육 콘텐츠 |

### 2.2 페르소나

**페르소나 1: 민준 (14세, 중학생)**
```
배경: 역사 과목이 지루하고 외울 게 많아서 싫어함
동기: 게임처럼 재미있게 역사를 알고 싶다
행동: 하루 30분 플레이, 친구와 발견한 내용 공유
결제: 부모님 승인 후 프리미엄 시대 구매
```

**페르소나 2: 서연 (28세, 직장인)**
```
배경: 여행 전 해당 지역 역사가 궁금함
동기: 출퇴근 시간에 가볍게 교양 쌓기
행동: 짧은 세션 (10-15분), 특정 지역만 집중
결제: 광고 제거 + 관심 지역 프리미엄
```

**페르소나 3: 김선생님 (42세, 초등교사)**
```
배경: 역사 수업을 더 흥미롭게 만들고 싶음
동기: 학생들에게 추천할 교육 앱 탐색
행동: 콘텐츠 정확성 검토, 학급 단위 추천
결제: 학교 라이선스 (B2B)
```

**시대 카드 상태**
```dart
enum EraStatus {
  locked,      // 🔒 미해금 - 회색, 흐릿함
  available,   // ⭐ 탐험 가능 - 빛나는 테두리
  inProgress,  // 🔵 진행 중 - 진행률 표시
  completed,   // ✅ 완료 - 황금 테두리, 별 표시
}

class EraCard {
  String name;           // "조선시대"
  String period;         // "1392-1897"
  String thumbnail;      // 시대 대표 이미지
  EraStatus status;
  double progress;       // 0.0 ~ 1.0
  List<Character> characters;  // 등장 인물
  List<Location> locations;    // 탐험 장소
  int estimatedMinutes;  // 예상 플레이 시간
}
```

**시대 해금 조건**
| 시대 | 해금 조건 |
|------|----------|
| 첫 번째 시대 | 기본 해금 (튜토리얼) |
| 두 번째 시대 | 첫 번째 시대 30% 완료 |
| 세 번째 시대 이후 | 이전 시대 50% 완료 OR 프리미엄 구매 |

```dart
class DialogueSystem {
  // 대화 노드
  class DialogueNode {
    String id;
    String speaker;           // 화자 (인물 ID)
    String emotion;           // 표정 상태
    String text;              // 대사 내용
    List<Choice> choices;     // 선택지 (없으면 자동 진행)
    String? nextNode;         // 다음 노드 (선택지 없을 때)
  }
  
  // 선택지
  class Choice {
    String text;              // 선택지 텍스트
    String preview;           // 선택 결과 미리보기
    String nextNode;          // 연결 노드
    Reward? reward;           // 획득 보상
    Condition? condition;     // 선택 조건 (특정 지식 필요 등)
  }
  
  // 대화 보상
  class Reward {
    int knowledgePoints;      // 역사 이해도 포인트
    String? unlockFact;       // 해금되는 역사 사실
    String? unlockCharacter;  // 해금되는 인물
    Achievement? achievement; // 획득 업적
  }
}
```
