# TimeWalker Portfolio Documents

TimeWalker 프로젝트의 포트폴리오 문서 모음입니다. 목적에 맞는 문서를 선택하여 사용하세요.

---

## 📚 문서 종류

### 1. [PORTFOLIO.md](./PORTFOLIO.md) - 상세 포트폴리오 (한국어)
**용도**: 기업 제출용, 상세 기술 설명, 블로그 포스트
**길이**: ~10 페이지 분량
**포함 내용**:
- ✅ 프로젝트 개요 및 핵심 가치
- ✅ 주요 화면별 상세 설명 (5개 화면)
- ✅ 기술 스택 완전 분석 (패키지별 용도)
- ✅ Clean Architecture 아키텍처 다이어그램
- ✅ 데이터 모델 및 콘텐츠 계층 구조
- ✅ 핵심 기능 상세 설명 (4개 시스템)
- ✅ 개발 프로세스 (5단계)
- ✅ 성능 최적화 전략
- ✅ 배포 전략 및 CI/CD 파이프라인
- ✅ 향후 개발 로드맵 (Phase 1-3)

**추천 대상**:
- Flutter 개발자 포지션 지원
- 게임/교육 앱 개발 경력 제시
- 기술 블로그/미디엄 포스트

---

### 2. [PORTFOLIO_EN.md](./PORTFOLIO_EN.md) - Detailed Portfolio (English)
**Usage**: International job applications, global companies, technical documentation
**Length**: ~10 pages
**Includes**:
- ✅ Project overview and core value propositions
- ✅ Detailed screen explanations (5 screens)
- ✅ Complete tech stack analysis
- ✅ Clean Architecture diagrams
- ✅ Data models and content hierarchy
- ✅ Core feature descriptions (4 systems)
- ✅ Development process (5 phases)
- ✅ Performance optimization strategies
- ✅ Deployment strategy and CI/CD pipeline
- ✅ Future roadmap (Phase 1-3)

**Recommended for**:
- International Flutter developer positions
- Global company applications
- English tech blog/Medium posts

---

### 3. [PORTFOLIO_SUMMARY.md](./PORTFOLIO_SUMMARY.md) - 1페이지 요약본 (이력서용)
**용도**: 이력서 첨부, 면접 자료, 빠른 프로젝트 소개
**길이**: 1-2 페이지 (A4 기준)
**포함 내용**:
- ✅ 프로젝트 기간, 역할, 기술 스택 요약
- ✅ 핵심 성과 6가지 (bullet points)
- ✅ 주요 기술 구현 (4가지 카테고리)
- ✅ 화면별 주요 기능 테이블
- ✅ 성능 최적화 요약
- ✅ 테스트 전략 다이어그램
- ✅ 개발 프로세스 타임라인
- ✅ 학습 및 성장 포인트
- ✅ 스크린샷 테이블 (5개)
- ✅ 기술 스택 한눈에 보기

**추천 대상**:
- 이력서 프로젝트 섹션 첨부
- 면접 시 자료 제시
- 빠른 프로젝트 개요 공유

   ```markdown

   - **Developer**: [Your Name]
   - **GitHub**: [Your GitHub URL]
   - **Email**: [Your Email]
   ```

2. **GitHub Repository 링크**:
   - README.md 상단 Clone 명령어
