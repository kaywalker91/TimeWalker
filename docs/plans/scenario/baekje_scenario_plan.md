## 💬 3단계: 대화 시나리오 설계

### 3.1 계백 시나리오: "황산벌의 마지막 전투"

**대화 ID**: `gyebaek_last_stand_01`  
**예상 소요 시간**: 10분  
**보상**: 지식 포인트 200, 업적 해금

#### 역사적 배경
- 660년, 나당연합군 13만이 백제 침공
- 김유신 5만 vs 계백 5천 (10:1 열세)
- 계백, 출정 전 처자식을 죽임 (배수진)
- 네 번 승리 후 관창의 희생으로 사기 오른 신라군에 패배

#### 스토리 흐름

```
[시작] 계백이 출정을 앞두고 비장한 결심
    │
    ├─[선택1] "왜 가족을 직접...?"
    │   └─> 비장함 분기 (결사의 각오, 눈물의 결단)
    │
    ├─[선택2] "5천으로 5만을 이길 수 있습니까?"
    │   └─> 전략 분기 (지형 이용, 결사대의 힘)
    │
    └─[선택3] "백제를 포기할 수는 없습니까?"
        └─> 충의 분기 (나라에 대한 충성, 멸망을 앞둔 비감)
            │
            └─> [결말] 네 번의 승리와 장렬한 패배
```

#### 핵심 대사 예시

```
"나는 오늘 이곳에서 죽을 것이다.
하지만 살아서 치욕을 당하느니, 싸우다 죽는 것이 백제 남아의 도리다.
내 처자식은 적의 손에 욕을 당하지 않을 것이야.
내가... 직접 보내주었으니까."

(눈물을 닦으며)

"울지 마라. 이것은 슬픔이 아니다.
이것은 결의다. 오늘 황산벌에서 우리의 이름을 역사에 새기자."
```

## 🔗 기존 콘텐츠와의 연계

### 근초고왕 ↔ 계백
- 근초고왕: 백제 최전성기 (4세기)
- 계백: 백제 멸망기 (7세기)
- 300년의 흥망성쇠 대비

### 의자왕 ↔ 무령왕
- 무령왕: 백제 중흥 (6세기 초)
- 의자왕: 백제 멸망 (7세기 중반)
- 중흥에서 멸망으로의 전환점

### 계백 ↔ 김유신
- 황산벌에서 직접 대결
- 관창의 희생 에피소드 공유
- 적이지만 서로 존경하는 라이벌

### 의자왕 ↔ 장수왕
- 장수왕: 백제 한성 함락, 개로왕 전사 (475년)
- 의자왕: 결국 신라에 멸망 (660년)
- 고구려-백제 갈등의 역사

## 🎭 감정적 톤

백제 시나리오는 다른 국가와 달리 **비극성**을 강조해야 합니다:

1. **근초고왕**: 영광과 자부심
2. **계백**: 비장함과 결의
3. **의자왕**: 후회와 비감
4. **무령왕**: 희망과 중흥

특히 계백의 시나리오는 플레이어가 눈물을 흘릴 수 있을 정도로 감동적으로 구성해야 합니다.
