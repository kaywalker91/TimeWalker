## 🎯 1단계: 미구현 대화 완성 (필수)

### 1.1 김수로왕: "운명의 만남" (`suro_hwangok_01`)

**배경**: 허황옥이 인도에서 도착한 직후, 수로왕과의 첫 만남  
**예상 소요 시간**: 7분  
**보상**: 지식 포인트 160

#### 핵심 주제
- 허황옥의 도착 상황
- 하늘이 정한 인연에 대한 확신
- 다른 문화에 대한 수용
- 왕비로서의 책임

#### 스토리 흐름

```
[시작] 수로왕이 망산도에서 배가 도착하는 것을 지켜본다
    │
    ├─[선택1] "저 배에서 누가 내립니까?"
    │   └─> 기다림 분기 (오래전부터 이 날을 기다렸다)
    │
    ├─[선택2] "이국의 공주를 왕비로 맞으시려고요?"
    │   └─> 열린 마음 분기 (하늘의 뜻, 다름을 포용)
    │
    └─[선택3] "신하들의 반대는 없습니까?"
        └─> 통치 분기 (왕의 결단, 새로운 피의 필요성)
            │
            └─> [결말] 첫 만남의 감동
```

#### 핵심 대사 예시

```
"짐은 꿈에서 이미 그 여인을 보았다.
붉은 돛을 단 배가 바다를 건너오는 것을.
하늘이 보낸 짝이라면, 
그 사람이 어느 땅에서 왔든 무슨 상관이겠느냐?

(망산도를 향해)

저기 온다... 
짐의 운명이 저 배에 타고 있다."
```

### 1.2 우륵: "신라에서의 새 삶" (`ureuk_exile_01`)

**배경**: 신라 진흥왕의 궁정, 망명 후 첫 만남  
**예상 소요 시간**: 8분  
**보상**: 지식 포인트 180

#### 핵심 주제
- 망명자의 심경
- 진흥왕과의 만남
- 제자들(계고, 법지, 만덕)과의 인연
- 예술의 영속성

#### 스토리 흐름

```
[시작] 우륵이 신라 궁정에서 가야금을 연주한다
    │
    ├─[선택1] "고국을 떠나셔서 힘드시겠습니다."
    │   └─> 향수 분기 (대가야에 대한 그리움)
    │
    ├─[선택2] "진흥왕께서 잘 대해주시나요?"
    │   └─> 후원 분기 (진흥왕의 음악 사랑, 국악원 설치)
    │
    └─[선택3] "제자들은 어떤 사람들입니까?"
        └─> 전승 분기 (계고·법지·만덕 소개, 각자의 재능)
            │
            └─> [결말] 음악은 나라를 초월한다
```

#### 핵심 대사 예시

```
"나는 더 이상 대가야의 신하가 아니오.
그러나 가야의 음악은 여전히 내 가슴에 살아 있다오.

진흥왕께서 말씀하시기를,
'그대의 음악은 가야의 것이 아니라 천하의 것이오.
신라에서 그 빛을 이어가라' 하셨다오.

계고는 가야금을, 법지는 노래를, 만덕은 춤을 배우고 있다오.
그들이 내 뜻을 이어갈 것이오.
나라는 망해도, 예술은 망하지 않는 법이니까."
```

## 🎯 2단계: 확장 대화 추가 (권장)

### 2.1 김수로왕: "철의 왕국" (`suro_iron_trade_01`)

**배경**: 낙동강변 철 교역소  
**예상 소요 시간**: 8분

#### 핵심 주제
- 가야 철의 우수성
- 왜, 중국과의 교역
- 경제 강국으로서의 가야
- 철이 가져다준 번영

#### 스토리 흐름

```
[시작] 수로왕이 철 거래 현장을 시찰하고 있다
    │
    ├─[선택1] "가야의 철은 무엇이 다릅니까?"
    │   └─> 기술 분기 (제련 기술, 품질의 비밀)
    │
    ├─[선택2] "왜에도 철을 수출합니까?"
    │   └─> 외교 분기 (왜와의 관계, 문화 전파)
    │
    └─[선택3] "고구려나 백제와는 경쟁하지 않습니까?"
        └─> 지정학 분기 (가야의 생존 전략, 연맹의 의미)
            │
            └─> [결말] 철이 곧 국력
```

#### 핵심 대사 예시

```
"이 철 덩이 하나가 쌀 열 섬과 맞먹는다.
가야의 힘은 군사가 아니라 이 철에서 나온다.

왜인들이 이 철을 구하러 바다를 건너오지.
중국 상인들도 우리 철만 보면 눈이 빛난다.
그들에게 철을 주면, 우리는 비단과 보석을 받는다.

이것이 가야의 방식이다.
칼로 빼앗는 것보다, 거래로 얻는 것이 현명하다.
그래서 우리는 작지만 강한 것이다."
```

### 2.2 허황옥: "어머니의 마음" (`hwangok_motherhood_01`)

**배경**: 왕궁 후원, 자녀들이 뛰노는 모습  
**예상 소요 시간**: 6분

#### 핵심 주제
- 10남 2녀의 어머니
- 두 아들에게 허씨 성을 줌
- 이방인에서 어머니로
- 후손들의 번영 기원

#### 스토리 흐름

```
[시작] 허황옥이 자녀들을 바라보며 미소짓고 있다
    │
    ├─[선택1] "12명의 자녀라니 대단하시네요."
    │   └─> 가족 분기 (각 자녀 소개, 왕자들의 역할)
    │
    ├─[선택2] "왜 두 아들에게 허씨 성을 주셨나요?"
    │   └─> 혈통 분기 (외가의 보존, 이중 정체성)
    │
    └─[선택3] "고향이 그립지 않으세요?"
        └─> 정체성 분기 (새 고향, 어머니의 땅)
            │
            └─> [결말] 나의 뿌리, 자녀들의 미래
```

#### 핵심 대사 예시

```
"열 아들과 두 딸...
모두 건강하게 자라주었어요.

두 아들에게는 제 성 '허(許)'를 주었어요.
제 조상들도 이 땅에 뿌리내리게 하고 싶었거든요.
나머지 아들들은 아버지의 성 '김(金)'을 이었고요.

훗날 누군가 허씨라고 하면,
그것은 먼 바다를 건너온 제 피가 흐르는 것이에요.
제가 없어도, 제 혈통은 영원히 이어지겠지요."
```

### 2.3 우륵: "12곡의 비밀" (`ureuk_12songs_01`)

**배경**: 대가야 궁정 연습실  
**예상 소요 시간**: 9분

#### 핵심 주제
- 12곡 각각의 의미
- 가야 지역들의 특색
- 음악으로 담은 역사
- 잃어버린 가야의 지도

#### 스토리 흐름

```
[시작] 우륵이 가야금을 손질하고 있다
    │
    ├─[선택1] "12곡은 모두 다른 지역을 담았나요?"
    │   └─> 지역 분기 (각 곡의 배경 설명)
    │
    ├─[선택2] "'상가라도'는 어떤 곡입니까?"
    │   └─> 상가라도 분기 (대가야의 노래, 가장 자랑스러운 곡)
    │
    └─[선택3] "가장 슬픈 곡은 무엇입니까?"
        └─> 비애 분기 (사물曲, 만물의 허망함)
            │
            └─> [결말] 12곡이 곧 가야의 역사
```

#### 핵심 대사 예시

```
"12곡은 가야의 12지역을 담았다오.

1. 하가라도 - 낙동강 하류, 금관가야의 노래
2. 상가라도 - 높은 산의 대가야
3. 보기 - 아라가야의 봄
4. 달이 - 달빛 아래 고령가야
5. 사물 - 만물이 덧없음을 노래함
...

(잠시 연주한다)

이 열두 곡을 모두 들으면,
가야 전역을 여행한 것과 같다오.
음악으로 그린 지도인 셈이지."
```
