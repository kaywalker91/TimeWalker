## 💬 3단계: 대화 시나리오 설계

### 3.1 을지문덕 시나리오: "살수대첩"

**대화 ID**: `eulji_salsu_01`  
**예상 소요 시간**: 10분  
**보상**: 지식 포인트 200, 업적 해금

#### 역사적 배경
- 612년, 수 양제가 113만 대군으로 고구려 침공
- 을지문덕의 청야전술(청야입벽): 들판을 비우고 성으로 들어감
- 수나라 별동대 30만 5천이 평양까지 진격
- 살수(청천강)에서 매복 공격으로 궤멸 (생존자 2,700명)

#### 스토리 흐름

```
[시작] 수나라 대군이 몰려온다는 급보
    │
    ├─[선택1] "적의 수가 너무 많습니다! 항복해야 합니까?"
    │   └─> 전략 분기 (청야전술 설명, 적을 지치게 하는 방법)
    │
    ├─[선택2] "정면으로 맞서 싸워야 합니다!"
    │   └─> 조언 분기 (손자병법 인용, 지혜로운 전쟁)
    │
    └─[선택3] "적장에게 시를 보내셨다던데..."
        └─> 시 분기 (여수장우중문시 낭송, 심리전)
            │
            └─> [결말] 살수대첩 승리, 수나라 30만 섬멸
```

#### 핵심 대사 예시

```
"하하, 그대는 싸움을 모르는구나.
군사가 많다고 이기는 것이 아니다.

우리는 집에서 싸우고, 저들은 객지에서 싸운다.

```

#### 여수장우중문시(與隋將于仲文詩) 활용

```

戰勝功旣高 (싸움마다 이겨 그 공이 높으니)
知足願云止 (만족을 알아 이제 그만 물러가시라)

- 을지문덕이 수나라 장수 우중문에게 보낸 시
- 조롱과 심리전의 걸작
```

## 🔗 기존 콘텐츠와의 연계

### 광개토대왕 ↔ 장수왕
- 장수왕은 광개토대왕의 아들
- 장수왕 대화에서 아버지에 대한 회상 포함
- 광개토대왕릉비 건립 스토리

### 광개토대왕 ↔ 근초고왕
- 이미 구현된 대화에서 백제와의 갈등 다룸
- 고국원왕(광개토대왕 증조부)이 근초고왕에게 전사한 역사

### 을지문덕 ↔ 연개소문
- 둘 다 외적 격퇴의 영웅
- 수나라 vs 당나라 격퇴 비교 가능
