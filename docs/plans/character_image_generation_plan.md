# Character Image Generation Plan

This plan outlines the image generation prompts for the newly added historical figures in TimeWalker. The goal is to create consistent, high-quality character portraits that fit the app's historical theme.

## 1. Unified Silla (통일신라)

| Character | Subject | Visual Prompts / Keywords | Mood/Style |
| :--- | :--- | :--- | :--- |
| **Jang Bogo** (장보고) | 'King of the Sea', Maritime Commander | Korean Silla dynasty general armor, commanding pose, ship deck background, ocean waves, sword at side, charismatic leader. | Heroic, Dynamic, Webtoon/Game Art Style |
| **Choi Chi-won** (최치원) | Genius Scholar | Silla official traditional robe (or Tang style), holding a brush and scroll, scholarly appearance, intelligent eyes, gentle but firm expression. | Intellectual, Serene, Portrait |
| **Wonhyo** (원효) | Eccentric Monk | Buddhist monk robes (worn out/humble), holding a gourd bottle, free-spirited smile, long hair (unconventional monk), nature background. | Free-spirited, Wise, Ruddy |

## Technical Specifications
- **Aspect Ratio**: 1:1 (Square) or 3:4 (Portrait) depending on UI usage (Default to 1:1 for profile icons).
- **Style Consistency**: Semi-realistic digital painting. Avoid pure photorealism; aim for a polished game asset look.
- **Lighting**: Dramatic but clear lighting on faces.
- **Background**: Simple, atmospheric backgrounds that hint at the era but don't distract from the character.
